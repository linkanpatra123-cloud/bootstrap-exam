Project Title : Freelancing Portfolio

Project Description:
This project is a modern and minimalistic personal portfolio webpage for a freelance developer named Dezven Indiana.
The layout is designed to immediately highlight the professional identity and create a strong first impression.
The main section introduces the individual, their role, and includes a call-to-action button, all supported by a stylish photo layout.


Key Features:
Responsive Navigation Bar with multiple section links (Home, About, Services, Experience, Portfolio, Testimonial, Contact)
Hero Section with developer's name, designation, and intro paragraph
Call-to-Action Button (“Contact Me”) styled using Bootstrap button classes
Circular Profile Image Frame with custom styling for enhanced visual appeal
Grid-based Layout using Bootstrap's row and col system
Typography and Spacing Utilities for clean layout and readability
Fully Responsive Design that works across desktops, tablets, and mobile devices
Minimalistic Color Theme using Bootstrap’s custom color utilities and extended styles


Technologies Used:
HTML5 for semantic layout
CSS3 for responsive styling and layout structuring
Bootstrap that make productive used
